Net Tic Toe is a CLI(Command Line Interface) Tic Tac Toe game programmed in Python 3.9
you will compete against an AI that will pick a random tile of 1 to 9.
The win condition are if you or the AI draw on 3 tiles that make a cross example XXX or X or X or X
